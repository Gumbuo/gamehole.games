# Game Hole

Browser games collection at gamehole.games

## Features

- **Alien Catacombs** - Top-down shooter game (Alpha Build)
- More games coming soon!

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

## Environment Variables

Create a `.env.local` file from `.env.local.example` and add your environment variables.

All sensitive API keys and secrets should be stored in `.env.local` which is never committed to git.

## Tech Stack

- Next.js 13 (App Router)
- TypeScript
- Tailwind CSS
- React 18
